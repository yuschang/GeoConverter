Beyond approach distance: A drone–hydrophone integrated framework for quantifying vessel noise masking risk to free ranging wild dolphins

![Graphical Abastract](https://github.com/user-attachments/assets/2cfda731-4540-4586-8ac7-283f43c78ca4)

Quantifying individual-level noise exposure for free-ranging marine mammals requires the fusion of heterogeneous, high-resolution data streams from multiple sensor modalities — a challenge that has not been addressed within a unified computational pipeline. This study presents an ecological informatics framework that integrates drone-based aerial video, YOLOv12 convolutional neural network-based vessel detection, a geoconversion algorithm (GeoConvertor) for pixel-to-geographic coordinate transformation using flight telemetry, and concurrent passive acoustic monitoring to estimate vessel-specific acoustic source levels and spatially explicit noise exposure at georeferenced animal positions. The pipeline was deployed in a coastal dolphin-watching setting off Jeju Island, South Korea, processing 187 vessel passage tracks across four vessel categories and 471 annotated Indo-Pacific bottlenose dolphin (Tursiops aduncus) surfacing events over 11 observation days. The GeoConvertor achieved sub-metric positional accuracy (mean error: 0.15 ± 0.08 m) and velocity estimation RMSE of 1.2–3.1 km/h at operational altitude. Linear regression of drone-derived vessel speed against hydrophone-measured source levels revealed strongly vessel-category-dependent relationships, with slope coefficients ranging from 0.16 to 1.77 dB/kt across frequency bands overlapping the dolphin whistle range (5.3–12.5 kHz). Dolphin-centric spatial analysis and threshold exceedance mapping showed that up to 93% of encounters exceeded a conservative 3 dB masking criterion, with vessel type — rather than proximity — emerging as the dominant predictor of noise exposure. The framework demonstrates how multi-sensor data fusion and deep learning-based object detection can generate ecologically actionable, individual-level noise impact assessments inaccessible to any single monitoring modality, and is transferable to other wildlife–vessel interaction contexts requiring spatially explicit exposure estimation.
